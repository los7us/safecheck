# PRIVACY POLICY

## SafetyCheck Privacy Policy

Last Updated: [Date]

### Data We Collect

**Submitted Content**:
- URLs and text you submit for analysis
- Stored temporarily in cache (1 hour) for performance
- Sent to Google Gemini API for analysis
- Not stored long-term

**Usage Data**:
- API request counts and timestamps
- Error logs (no personal information)
- Performance metrics

**Technical Data**:
- IP addresses (in server logs only, not stored)
- Browser/client information (standard HTTP headers)

### How We Use Data

- Analyze content for safety risks
- Improve service performance (caching)
- Monitor system health and errors
- Prevent abuse (rate limiting)

### Data Sharing

- **Google Gemini**: Content sent for AI analysis (see Google's privacy policy)
- **No other third parties**: We do not sell or share data

### Data Retention

- **Cache**: 1 hour maximum
- **Logs**: 30 days maximum
- **Metrics**: Aggregated, no personal data

### Your Rights

- **Right to deletion**: Clear cache via API
- **Right to access**: Contact us for data access
- **Right to opt-out**: Don't use the service

### Security

- HTTPS encryption in transit
- No long-term storage of content
- API key authentication
- Regular security audits

### Contact

For privacy concerns: privacy@safetycheck.app
